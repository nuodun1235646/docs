## 品定价和销售逻辑

跟单猿的产品目前分为3档：免费、vip、私有化

|    | 免费                          | VIP                                      | 私有化                                |
| --- | --------------------------- | ---------------------------------------- | ---------------------------------- |
| 介绍 | 在用户承担一定的网络成本后，可免费使用普通的跟单功能。 | 支付月费，可使用跟单猿全部的功能                         | 起步1500u一年，永久VIP，可独立部署交易服务，享受更稳定的交易 |
| 限额 | 2000u                       | 5000u起步，最高只能到2万。突破2万必须私有化。               | 5万u起，每增加5万u额度，售价提高500u             |
| 作用 | 尝鲜体验，不做推荐。起锚定作用             | 作为筛选潜在年费或者私有化客户的工具。首推VIP，然后将月付VIP转为年付VIP | 软件销售利润最大化，从VIP中筛选大资金客户，推荐私有化部署版版本  |
| IP | 8人共享                        | 8人共享/付费独享<br />独享IP额外支付80u一年             | 独享，服务器一年                           |

对于IP和额度限制的解释：

1. 5000u基本上已经可以满足80%的散户，如有更多资金额度的需求，说明这个客户是潜在年费或者私有化的客户。
2. 交易所对一个API单笔交易金额有上限限制，如果超额，将导致无法交易成功。在资金达到2万u时，如果前期多次加仓，最后一次性平仓，就会很容易出现这种情况。为了避免大额仓位无法平仓，就必须拆分多笔小额交易，然后并发执行。
3. 交易所同时对IP交易频率有限制，比如2秒内20笔交易。一般情况下，8人共享一个ip交易是不会出现拥堵的。但是如果有2个人同时跟单了同一个高频量化，就会很容易出现交易频率过高。此时就需要考虑独享IP。而资金超过2万u的客户，由于可能会需要拆分并发平仓，就更加需要有一个独享IP，鉴于他的资金量完全可以考虑私有化交易服务。

| 产品    | 售价                     |
| ----- | ---------------------- |
| 共享IP  | 自定义，不得低于15u            |
| vip   | 自定义，不得低于30u            |
| 年费vip | vip价格 \* 12 \* 0.9     |
| 代理资格  | 年费vip自动获得              |
| 私有化   | 1500u起，每\+5万u额度，\+500u |
| 独享IP  | 80u一年                  |

以上价格会根据情况定期调整

## 跟单猿目前状况介绍

跟单猿的运营成本非常低，相较于同行竞品，成本仅为1/5甚至1/10。但是服务器的稳定性相较于亚马逊云、谷歌云、阿里云还是有点差距，运营商容易遭受网络攻击，导致我们会被动服务终端。

这一缺点并非不可克服，稳定性一直是我们追求的最终目标。我们已经做好了产品分布式架构，未来用户量起来了，我们有了资金后会在多个国家和地区部署服务器，如果监控到主服务器宕机后，会立刻切换到其他地区进行服务，切换过程只需30秒。

跟单猿从开发上线之初就一直是自给自足，以战养战状态，并没有过多投入资本，相信未来一定会很快实现多服务器运营。但是在此之前，还请做好心理准备，遇到问题时多和客户沟通解释。表明我们只是辅助工具，运营成本很高，受到网络攻击或者网络波动的影响也是无法避免的，即使是阿里云也会遭受攻击影响。遇到服务器问题，技术这边现在也会在第一时间通过消息通知和邮件进行通知，告知用户留意仓位变化。

一般情况下服务不会出现太大问题，如有问题可以及时沟通，如果对某个交易任务有疑问，需带上任务id方便查看日志分析。
